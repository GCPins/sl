### The original `sl` command with some of my own personal changes.

**Available Flags:**\n
[q/Q] - The locomotive can move faster\n
[s/S] - The locomotive can move slower\n
[e/E] - The program can be quit\n

Help menu/command info:

```sh
man ./sl.1
```

OR

```sh
./sl -h

# alternatively:
# ./sl -H
```
