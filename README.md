### sl2

The original `sl` command with some of my own personal changes.

**Available Flags:**<br/>
[q/Q] - The locomotive can move faster<br/>
[s/S] - The locomotive can move slower<br/>
[e/E] - The program can be quit<br/>

Help menu/command info:

```sh
man ./sl.1
```

OR

```sh
./sl -h

# alternatively:
# ./sl -H
```
